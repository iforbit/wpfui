# Project Dev Log (WPF.UI.DirectX)

## 📅 2025-06-22: Series 아키텍처 통합 설계 완료

### ✅ 아키텍처 전환 방향

* 기존 `GraphItemBase` 구조를 단계적으로 폐기하고 `GraphSeries<T>`를 중심으로 한 데이터 기반 구조로 전환 결정
* 모든 시리즈는 `IGraphSeries` 또는 `GraphSeries<T>`를 상속하여 구현하며, 실시간 시리즈는 `RealTimeSeries<T>`로 통일
* 내부 요소(Axis, Cursor, Marker 등)도 시리즈로 처리하고 `DrawKey` 기준으로 병합 렌더링될 수 있도록 설계
* `DrawCall` 최소화를 위한 `DrawGroupBatcher` 구조 도입 계획 확정
* 렌더링 파이프라인은 `SeriesRendererManager` 중심으로 구성되며, `SeriesRendererBase<T>`가 실질적 GPU 처리를 담당

### 🔧 기술 방향 확정

* `ViewMatrix`, `ProjectionMatrix`는 GPU 상에서 처리하며 Transform 설정은 시리즈 내부에 위임
* 각 시리즈는 렌더러에 의해 GPU 자원이 자동으로 초기화되며, WPF와 DirectX의 병목 없이 독립적으로 렌더링됨
* 시리즈와 렌더러는 `_seriesList`를 기준으로 매핑 및 호출되어 GraphControl과 Renderer 간 명확한 역할 분리 유지
