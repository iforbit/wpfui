## 📅 2025-06-26

### ✅ 오늘의 주요 작업 내역

#### 🧩 RealTimeSeries 구조 안정화 및 렌더링 통합 점검

- `RealTimeSeries<T>` 시리즈에서 정렬 미적용을 기본 구조로 확정
  - 실시간 수신 순서대로 `Append()`되며, `LineStrip` 방식으로 자연스럽게 그려짐
  - 데이터 순서가 일부 어긋나도 렌더링 품질에 영향 없음
  - 추후 "정밀 보기 모드"나 저장된 데이터 뷰어를 위해 `GetSortedSpan()`, `EnableDrawSort` 등의 옵션은 별도 설계로 보존

#### 🔁 SharedMemory 최적화 통합

- SignalEngine에서 생성한 데이터를 실시간으로 수신 및 렌더링하는 구조 개선
- `MemoryMappedViewAccessor`의 포맷 확장을 통해 한 틱당 다수의 포인트를 연속 수신
- `Span<T>` 기반 `Append()` 호출로 GC 최소화 및 고속 처리 실현

#### 🧭 TransformManager와 X축 시간 확대 대응 준비

- `TransformManager.XScale`을 통한 시간축 확대 시 `μs` 단위까지 표현 가능하도록 설계 방향 확정
- 향후 구현될 `AxisManager`에서 다음 요소 적용 예정:
  - 시계열 축 레이블 간격 자동 계산
  - 확대 레벨에 따라 `μs`, `ms`, `s`, `min` 단위로 유동적 포맷 적용

#### 🔀 렌더링 파이프라인 연계 점검

- 렌더링 흐름: `GraphControl → D3D11Renderer → DrawGroupBatcher → SeriesRenderer` 정상 작동 확인
- `DrawGroupBatcher`를 통한 DrawCall 병합으로 성능 최적화 확인
- `RealTimeSeriesRenderer<T>`는 `GetSpanMergedUnsafeCached()`를 통해 wrap-around된 데이터를 끊김 없이 병합 후 렌더링
